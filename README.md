# ⚡ Grid Commander: Heatwave Survival

**Can you keep the lights on during a heatwave?**

Grid Commander: Heatwave Survival is an interactive browser-based
strategy game where you take control of a city's power grid during
extreme weather conditions.

## 🎮 Gameplay

Your mission is to keep the city powered while temperatures soar.

You'll need to make strategic decisions:

-   ☀️ Deploy renewable energy at the right time.
-   🔋 Manage battery storage wisely.
-   ⚡ Balance electricity supply and demand.
-   🏙️ Prevent the city from going dark.

It sounds simple---until demand spikes and every decision matters.

## 🌍 About the Game

This simulation is inspired by real-world challenges faced by modern
power grids as renewable energy, battery storage, and increasing
electricity demand reshape energy management.

## 🏆 Challenge

How many days can you keep the grid running before a blackout occurs?

Try different strategies to maximize your survival time and keep the
grid stable.

If you improve the game or discover an effective strategy, feel free to
share your results!

## 🚀 Features

-   Interactive 3D visualization
-   Real-time grid monitoring
-   Renewable energy management
-   Battery storage simulation
-   Dynamic heatwave events
-   Strategy-based gameplay

## 🛠️ Technologies Used

-   HTML5
-   CSS3

-   Three.js

## ▶️ Run Locally

1.  Clone this repository.
2.  Open `index.html` (or the game HTML file) in a modern web browser.
3.  Start managing the grid!
